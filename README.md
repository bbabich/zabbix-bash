zabbix-bash
===========

A Bash wrapper for the [Zabbix API](https://www.zabbix.com/wiki/doc/api).

Requirements
------------

+ bash
+ curl

Getting Started
---------------

```bash
zabbix.sh user login --user admin --password zabbix
```

Usage
-----

```bash
zabbix.sh <namespace> <command>
```

Installation
------------

```bash
git clone https://github.com/hansode/zabbix-bash
```
